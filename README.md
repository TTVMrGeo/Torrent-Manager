# Setup

Clone repo:

```git clone https://github.com/TTVMrGeo/Torrent-Manager.git```

Install requirements:
```
# If on windows or mac:

cd Torrent-Manager
pip install flet

# If on linux:

cd Torrent-Manager
python3 -m venv env
source env/bin/activate
pip install flet
```

Running the app:

```
# WHILE IN THE PROGRAM FOLDER

# Windows:
python app.py

# Linux:
source env/bin/activate
python app.py
```
